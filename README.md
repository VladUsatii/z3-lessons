# Z3 Lessons

Written by Vlad Usatii @ RIT to better understand frontier Z3 for research applications.

There are, as you may soon see, an incredible amount of ways to apply this technology to your everyday life. We will only look at a subset of those use cases.

### Important notes

* The ```.venv``` should be constructed at this root so that you don't need to re-build on every lesson.
* You should supplement with this [paper](https://people.eecs.berkeley.edu/~sseshia/pubdir/SMT-BookChapter.pdf).
* If you want to trace your Z3 to see what is happening under the hood, please refer to ```harness.py```.
